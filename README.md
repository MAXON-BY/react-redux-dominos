
## Используемые команды

Для старта проекта запускаем сервер

### `npm run server`

Для запуска реакта

### `npm run start`


## Используемые библиотеки

- react
- react-dom
- axios
- classnames
- json-server
- node-sass
- prop-types
- react-spinners
- redux
- react-redux
- redux-thunk
- react-router-dom
- react-slick

Используется proxy: http://localhost:3001
